> **"Before you defend systems, you must understand how they are built."**

This repository provides a structured, multi-phase roadmap for mastering computer architecture, operating systems, and troubleshooting—essential for IT Support, System Administration, and Cybersecurity.

---

## 🚀 Roadmap Overview

### 📍 Phase 1: Hardware & Architecture
* **CPU/GPU:** Logic gates, ALU, clock speed, and multithreading.
* **Memory & Storage:** SRAM vs. DRAM, Paging, NVMe, and NAND flash logic.
* **I/O Systems:** How peripherals communicate with the Kernel.

### 📍 Phase 2: OS & Virtualization
* **Kernel Internals:** Process management, system calls, and file systems (NTFS/ext4).
* **Virtualization:** Type-1 vs. Type-2 Hypervisors and Docker containerization.

### 📍 Phase 3: Networking & Linux
* **Core Networking:** TCP/IP stack, DNS, DHCP, and Subnetting.
* **Linux Mastery:** Bash scripting, permissions, and terminal-based system administration.

### 📍 Phase 4: Cybersecurity & Automation
* **Security Foundations:** CIA Triad, Malware analysis, and Defensive/Offensive paths.
* **Python:** Automation scripts for log parsing and API interaction.

---

## 🛠️ Career Pathways
Completion of this roadmap prepares you for:
* **IT Helpdesk / Technical Support Engineer**
* **Junior System Administrator**
* **Cybersecurity Analyst (SOC Level 1)**
* **Hardware Specialist**

## 📖 How to Use This Roadmap
1.  **Theory:** Follow the PPTs and conceptual videos linked in each phase.
2.  **Lab:** Build virtual machines and practice Linux commands.
3.  **Troubleshoot:** Learn by breaking and fixing your own local environments.
